## Let's start with the Demo

```js
const uuid = () => {
    let url = URL.createObjectURL(new Blob())
    return URL.revokeObjectURL(url) || url.split('/')[3]
}
// or

const uuid = (url = URL.createObjectURL(new Blob())) => (
    URL.revokeObjectURL(url) || url.split('/')[3]
)
```
